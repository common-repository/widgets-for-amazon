== Description ==

**Widgets for Amazon** offers search box widgets that were previously offered by Amazon (they shut down these widgets in April 2022).

### Features

- Offers a very similar look and feel as old Amazon search widgets.

### Contributing

You can contribute to **Widgets for Amazon** in different ways. For example:

- Write about the plugin on your blog or share it on social media.
- [Vote on features in the GitHub issue list](https://github.com/eggnstone/widgets-for-amazon/issues).

### Pricing

* **Free version (this)**
  The free version comes at no cost for you. You can use it as much as you want.
  Please note that one tenth of the search results will be using our tag in order to support the development of the plugin and to cover the server costs on our side.
* **Paid versions**
  We're working on a paid version for higher traffic sites.
  Contact us at [widgets-for-amazon-wordpress@eggnstone.com](mailto:widgets-for-amazon-wordpress@eggnstone.com) if you are interested to become an early adopter and get special prices.

== Installation ==

1. In your WordPress admin area, go to **Plugins > New Plugin**, search for **Widgets for Amazon** and click **Install now**.
2. Activate the plugin.
3. Go to the plugin settings and review the settings.
4. Add shortcodes to your pages and posts, e.g. <code>[amazon-search keywords="Toys"]</code>
